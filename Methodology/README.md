# Methodology Documents

This folder contains draft methodology papers for the REAL-AI-Benchmark project.

## Current documents

- `GO_benchmark_standardization_methodology_GO5_extended.pdf` — Serbian methodology draft for GO-1 to GO-5.
- `GO_benchmark_standardization_methodology_GO5_extended_EN.pdf` — English methodology draft for GO-1 to GO-5.
- `GO-6_fizicki_AI_benchmark_metodologija.pdf` — Serbian methodology draft for GO-6 physical-AI benchmarking.
- `GO-6_Physical_AI_Benchmark_Methodology_EN.pdf` — English methodology draft for GO-6.

## Citation note

These are public research drafts. When using or discussing the benchmark methodology, cite the repository and the relevant methodology document. A repository-level `CITATION.cff` file is included for GitHub citation metadata.
