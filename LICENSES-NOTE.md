# Licensing Model

The REAL-AI-Benchmark repository uses a single repository-wide license:

**Creative Commons Attribution 4.0 International (CC BY 4.0).**

This replaces the earlier GPL-3.0 license file. The project is primarily an
academic benchmark and methodology repository, containing benchmark prompts,
methodological documents, diagrams, figures, result logs, scoring rubrics, and
reference snippets. A single CC BY 4.0 license keeps the project simple,
citation-friendly, and easy to reuse with mandatory attribution.

Unless explicitly stated otherwise, all repository contents are licensed under
CC BY 4.0.
