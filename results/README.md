# Results

This folder is intended for raw model outputs, normalized result tables, and generated plots.

Recommended structure:

```text
results/
├── raw-logs/
├── tables/
│   ├── results_template.csv
│   └── result_schema.json
└── plots/
```

Raw logs should be preserved without silent correction. Evaluator comments, repaired code, and normalized scores should be stored separately.
