# Sponsorship

REAL-AI-Benchmark is an independent practical AI model evaluation project focused on real-world task performance instead of synthetic leaderboard appearance.

## Why sponsorship matters

Running and documenting these evaluations requires:

- high-RAM local inference machines,
- GPUs and accelerators,
- SSD/NVMe storage for large GGUF models,
- electricity and cooling,
- paid API/model access,
- repeated runs for robustness,
- manual verification of mathematical reasoning and code,
- charting, reporting, and public documentation.

## What sponsorship supports

Sponsorship helps fund:

1. broader model coverage,
2. repeated benchmark reruns,
3. improved result-extraction tooling,
4. public Excel/CSV result tables,
5. reproducible reference solutions,
6. local large-model testing,
7. long-context and multimodal benchmark development.

## Sponsorship channels

Planned or suitable support channels:

- GitHub Sponsors,
- institutional support,
- hardware donation,
- cloud/API credits,
- storage or hosting support,
- collaboration on benchmark validation.

Until GitHub Sponsors is configured, open an issue titled `Sponsorship / Support` or contact the maintainer through the repository profile.
